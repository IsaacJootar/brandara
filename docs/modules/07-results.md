# Module 07 — Results (Analytics & Intelligence)

## Purpose
Clean, actionable analytics. What is working, on which platform, and why.
Not vanity metrics — decision-driving data.

## Screens
- `/results` — main dashboard with overview metrics
- `/results/content` — post performance detail
- `/results/platforms` — per-platform breakdown
- `/results/pillars` — pillar analytics
- `/results/health` — platform connection health

## Content performance dashboard

**Stat cards (each unique colour — see colors.md):**
- Total posts published this month (violet)
- Total reach across all platforms (blue)
- Warm leads from engagement this week (amber)
- Average engagement rate (rose)
- Active campaigns running (teal)
- WhatsApp messages generated (orange)

**Post performance table:**
- Engagement rate, impressions, reach per post
- Sorted by performance (best first)
- Filterable by platform, pillar, date range, campaign
- Click any post → detail view with full engagement breakdown

**Best performing formats:**
- Carousel vs text post vs quote graphic comparison
- Which format drives most saves, shares, comments

**Best posting times:**
- Heatmap of engagement by day and hour
- "Your audience engages most on Tuesdays at 9AM WAT"

## Platform comparison
Side-by-side metrics per platform:
- Engagement rate
- Reach
- Post frequency
- Follower growth trend (last 30 days)
- Where to focus more effort

## Pillar analytics
- Engagement rate per content pillar
- Posting frequency vs performance correlation
- "Your Thought Leadership posts get 3x more saves — post more of these"
- Pillar gap alerts: X days since last [pillar] post

## Platform health monitor
- Status indicator per connected platform (green/amber/red)
- Last successful publish per platform
- Token expiry countdown with reconnect prompt
- Failed post count with link to Fix & Retry
- API rate limit status

## AI Reporting Summaries
Weekly (Monday 8AM) and monthly narrative reports.
Generated by Claude. Sent via email + web push + in-app.

Weekly summary example:
"Your best post this week was [Title] on LinkedIn with 420 engagements.
Your strongest day was Tuesday at 9AM. You haven't posted on Client Wins
in 10 days — here are 3 ideas. Ramadan starts in 14 days — start your campaign now."

Monthly health report includes:
- Pillar balance score
- Platform consistency
- Top 3 posts
- Growth trends
- Actionable next steps
- Shareable as PDF or URL (for agency client reports)

## Data sources
- Analytics pulled from platform APIs daily (LinkedIn, X, Meta)
- Stored in posts table after enrichment
- Cache analytics data — refresh every 24 hours max
- Show last_updated timestamp on dashboard

## Database tables
- `posts` — engagement data stored per post on sync
- `platform_connections` — follower counts updated periodically
